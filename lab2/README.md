# RSA на tkinter

Учебное приложение демонстрирует работу асимметричного алгоритма шифрования RSA без использования сторонних библиотек.

Реализовано:
- генерация пары ключей RSA;
- шифрование открытым ключом;
- расшифрование закрытым ключом;
- графический интерфейс на `tkinter` по мотивам предоставленного макета.

## Запуск

```bash
python main.py
```

## Как пользоваться

1. Введите исходный текст в блоке «Шифрование».
2. Нажмите «Зашифровать».
3. Полученный шифртекст появится в блоке «Шифр».
4. Нажмите «Расшифровать», чтобы восстановить текст в блоке «Расшифровка».
5. Для создания новой пары ключей используйте кнопку «Новые ключи» или пункт меню «Файл -> Сгенерировать ключи».

## Особенности реализации

- Программа самостоятельно генерирует простые числа и вычисляет ключи RSA.
- Для проверки простоты используется тест Миллера-Рабина.
- Исходный текст кодируется в UTF-8, поэтому поддерживаются русские символы.
- Шифртекст отображается как последовательность целых чисел, разделённых пробелами.

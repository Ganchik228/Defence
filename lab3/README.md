# Лабораторная работа 3: Хеш-функция MD5

## Цель
Исследование структуры алгоритма хеширования MD5 и программная реализация полной функции хеширования.

## Что реализовано
- Полная реализация MD5 без использования `hashlib` в вычислении хеша.
- Полный цикл алгоритма:
	- дополнение сообщения (padding);
	- добавление длины сообщения;
	- обработка блоков по 512 бит;
	- 64 раунда преобразований с константами и циклическими сдвигами;
	- формирование итогового 128-битного хеша.
- Графический интерфейс на `tkinter`:
	- ввод текста;
	- выбор кодировки (`utf-8`, `utf-16`, `utf-16le`, `utf-16be`, `cp1251`);
	- кнопка вычисления хеша;
	- поле результата и копирование в буфер обмена.

## Запуск
1. Активировать виртуальное окружение.
2. Запустить:

```powershell
python main.py
```

## Структура
- `main.py` — реализация алгоритма MD5 и GUI-приложение.

## Примечание
Для проверки корректности реализации можно сравнивать результат с `hashlib.md5(...).hexdigest()` на тестовых строках.
